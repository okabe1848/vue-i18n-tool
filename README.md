# vue-i18n-tool
## How to run
- Copy the contents of the target vue file (with Japanese) in /ja.txt
- Run this app
    - ./run-tool.sh
- Extract Japanese and list up in your Spread Sheet
    - ./extract.sh
- Edit your Spread Sheet
- Convert vue file from JA to EN
    - ./convert.sh
- Insert dictionary
    - ./dictionary.sh
    - ./dictionary.sh
